# 品質チェックリスト追加分 — Webアプリ

> Layer-0（共通）+ Layer-1（言語別）のチェックリストに**追加**する項目のみ。

## Webアプリ固有の品質基準

### アクセシビリティ（a11y）
- [ ] キーボードのみで全機能が操作可能
- [ ] 適切な ARIA ラベルが設定されている

### パフォーマンス
- [ ] Lighthouse Performance スコアが 90+
- [ ] バンドルサイズが適切（不要な依存がない）

### セキュリティ
- [ ] XSS 対策: ユーザー入力がエスケープされている
- [ ] APIキーは暗号化保存されている
- [ ] CORS制限が適切に設定されている

### E2Eテスト
- [ ] Playwright テストが主要ユーザーフローをカバー
- [ ] APIキー設定→ダッシュボード表示のフローテスト

### レスポンシブ
- [ ] モバイル / タブレット / デスクトップで表示確認
